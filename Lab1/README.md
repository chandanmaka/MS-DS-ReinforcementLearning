# Week 1 Reinforcement Learning Lab

This folder contains the Week 1 assignment notebook:

- `Week_1_assignment.ipynb`

## Requirements

Install dependencies from `requirements.txt`:

```bash
pip install -r requirements.txt
```

## Running the Notebook

Start Jupyter and open the notebook:

```bash
jupyter notebook
```

Then open `Week_1_assignment.ipynb` in your browser.
