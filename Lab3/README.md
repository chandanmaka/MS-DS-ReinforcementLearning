# Lab 3 - Reinforcement Learning

This folder contains the Week 3 reinforcement learning assignment notebook.

## Files

- `Week_3_assignment.ipynb`: Main notebook for Lab 3.
- `requirements.txt`: Python dependencies used by the notebook.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

Open `Week_3_assignment.ipynb` in Jupyter Notebook or VS Code and run the cells in order.
