# Lab 4 — Week 4 Assignment

Reinforcement learning exercises in `Week_4_assignment.ipynb` (Gymnasium Cliff Walking and related work).

## Setup

Use Python 3.10 or newer (3.11+ recommended).

```bash
cd Lab4
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Run the notebook

```bash
jupyter notebook Week_4_assignment.ipynb
```

Or open the notebook in VS Code / Cursor with a Python interpreter that has these packages installed.

## Dependencies

See `requirements.txt`. Core packages: **Gymnasium**, **NumPy**, **Matplotlib**, **Pandas**, and **Jupyter** for the notebook UI.
