# Lab 6 - Reinforcement Learning

This folder contains materials for the Week 6 Reinforcement Learning assignment.

## Contents

- `Week_6_assignment.ipynb`: main notebook for the lab
- `requirements.txt`: Python dependencies for running the notebook

## Setup

1. Create and activate a virtual environment:
   - macOS/Linux:
     ```bash
     python3 -m venv .venv
     source .venv/bin/activate
     ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start Jupyter:
   ```bash
   jupyter notebook
   ```
4. Open `Week_6_assignment.ipynb` and run the cells.

## Notes

- Use Python 3.9+.
- If a package is missing while running the notebook, add it to `requirements.txt`.
